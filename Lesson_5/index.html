JQuery Example Project - Lesson 5

This is the sample project for lesson 5, which doubles as a cheat-sheet for what we learn in this lesson

The "hide" method

            
                $('#my-element').hide(
                    1000,
                    'swing',
                    function() {
                        console.log("Animation completed!");
                    },
                );

This method hides the selected element or elements with a basic animation. Using the arguments, you're allowed to specify how long this animation will take, the basic acceleration of the animation, and a callback function that will be called when the animation completes.

The "show" method

            
                $('#my-element').show(
                    1000,
                    'swing',
                    function() {
                        console.log("Animation completed!");
                    },
                );

This method does the opposite of the hide method, it reveals a hidden element using an animation. The arguments here are the same as with "hide".

The "toggle" method

            
                $('#my-element').toggle(
                    1000,
                    'swing',
                    function() {
                        console.log("Animation completed!");
                    },
                );

This method is a combination of "hide" and "show". It animates a hidden element onto the page and hides a visible element.

The "animate" method

            
                $('#my-container').animate({
                    padding: '16px',
                    maxWidth: '400px',
                });

Use this method to animate numeric CSS attributes. For example, we can animate an element getting wider, an element getting more padding, etc. By default, this method doesn't work with colors, but you can add this functionality using a library called jQuery UI. Note that CSS attribute names in the configuration object should be in camelCase.

The "delay" method

            
                $('#my-element')
                    .delay(1000)
                    .hide();

Use this method to delay the execution of an animation, or to put a "rest" between subsequent animations.

The "queue" method

            
                $('#my-element')
                    .hide()
                    .delay(1000)
                    .show()
                    .animate({ width: '500px' });

                $('#my-element').queue();

Use this method to get a list of all unexecuted animations for a given element.

The "stop" method

            
                // Pause an element's animation
                $('#my-element').stop();
            
            
                // Stop and clear an element's animations
                $('#my-element').stop(true);
            
            
                // Clear and jump to the end of an element's animations
                $('#my-element').stop(true, true);

Use this method, with varying combinations of arguments, to pause, clear, or jump to the end of an element's animation queue (respectively).

The "fadeOut" method

            
                $('#my-element').fadeOut();

Use this method to fade a visible element off the page. It's a shortcut for fading the opacity to 0 and then setting "display" to "none".

The "fadeIn" method

            
                $('#my-element').fadeOut();

Use this method to fade a hidden element back on to the page. It's a shortcut for setting "display" to "none", and then fading to full opacity.

The "fadeToggle" method

            
                $('#my-element').fadeToggle();

This method is a combination of fadeIn and fadeOut. It will fade an element onto the page if it's hidden and fade it off the page if it's visible.

The "fadeTo" method

            
                $('#my-element').fadeTo(0.5);

Use this method to fade an element to a given opacity. The argument given should be from 0 (completely transparent) to 1 (completely opaque).

The "slideUp" method

            
                $('#my-element').slideUp(400)

This method will slide an element up and then make it disappear from the page. Basically, it reduces an element's height to 0. We can pass an optional argument that determines how long in milliseconds the animation should last.

The "slideDown" method

            
                $('#my-element').slideDown(400)

This method will slide a hidden element into its spot on the page. Basically, it animates an element's height from 0 to whatever its appropriate height should be. We can pass an optional argument that determines how long in milliseconds the animation should last.

The "slideToggle" method

            
                $('#my-element').slideToggle()

This is a combination of slideUp and slideDown, and will behave like either one depending on if the element is currently visible or hidden.

The "height" and "width" methods

            
                $('#my-element').height();
            
            
                $('#my-element').width();

These methods return the height or width, in pixels, of a given element.

The "innerHeight" and "innerWidth" methods

            
                $('#my-element').innerHeight();
            
            
                $('#my-element').innerWidth();

These methods return the inner height or width, in pixels, of a given element. The "inner height/width" is the size of an element's content, plus the size of the padding.

The "outerHeight" and "outerWidth" methods

            
                $('#my-element').outerHeight();
            
            
                $('#my-element').outerWidth();

These methods return the outer height or width, in pixels, of a given element. The "outer height/width" is the size of an element's content, padding, and border. We can pass an optional argument to specify whether or not an element's margin should be included in the measurement.
