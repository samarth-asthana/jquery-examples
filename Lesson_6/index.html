JQuery Example Project - Lesson 6

This is the sample project for lesson 6, which doubles as a cheat-sheet for what we learn in this lesson

The "find" method

            
    $('.todo-list').find('li')

This method finds all elements that match a given selector that lie inside a previously-selected element.

The "children" method

            
    $('#info-div').children();

This method gets all the elements that are contained inside the selected element.

The "parent" method

            
    $('#my-element').parent();

This method gets the DOM element that is surrounding the selected element.

The "parents" method

            
    $('#my-element').parents();

This method gets the DOM element that is surrounding the selected element, as well as the one surrounding THAT element, and so on, all the way up to the html element that surrounds all the elements on the page

The "siblings" method

            
    $('#some-heading').siblings();

This method returns all the other direct children of the selected element's parent element. In other words, it returns all other elements inside the parent on the same level as the selected element.

The "each" method

            
    $.each( $('h3'), function(i, element) {
        console.log($(element).text());
    });

This method allows you to perform some sort of action for each element in a selection, for example, printing the inner text of all h3 headings.

The "filter" method

            
    $('input').filter(function(i, element) {
        return $(element).val().length === 0;
    }).css(‘border’, ‘1px solid red’);

This method allows us to filter a selection programmatically, instead of by matching some selector. For example, we can use it to apply a red border to all input elements that don't contain any text.

The "has" method

            
    $('div').has('h3');

This method is used to filter a selection of elements based on whether or not those elements contain some other element that matches a given selector. For example, finding all divs on the page that contain an h3 element.

The "is" method

            
    $(‘button’).click(function() {
        if( $(this).is(‘:even’) ) {
            console.log(“I’m even!”);
        } else {
            console.log(“I’m odd!”);
        }
    });

Use this method to find out programmatically whether or not a given element matches a given selector (this method returns a boolean). This can be used, for example, to figure out whether a given element is in an even-indexed position on the page and apply some sort of styling or custom behavior to it.
